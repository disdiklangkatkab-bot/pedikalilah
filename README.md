## Tentang Aplikasi

PEDIKALILAH adalah aplikasi yang dikembangkan untuk membantu satuan pendidikan melakukan pencegahan dini terhadap kasus bullying di sekolah melalui sistem pelaporan dan monitoring yang mudah digunakan.

## Fitur

- 📋 Pelaporan kasus bullying
- 📊 Monitoring laporan
- 📚 Edukasi anti bullying
- 🔔 Notifikasi
- 🏫 Mendukung lingkungan sekolah yang aman
